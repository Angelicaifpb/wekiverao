Apresentação do Projeto

Esta página faz parte de um projeto desenvolvido para a disciplina de Desenvolvimento Web, no curso Técnico em Informática. A proposta é aplicar os conhecimentos básicos de HTML para construir uma página informativa sobre uma obra audiovisual de destaque.

O tema escolhido para esta atividade foi a série “O Verão que Mudou a Minha Vida”, que tem conquistado muitos fãs ao redor do mundo por sua abordagem sensível sobre o crescimento, as descobertas da adolescência e os dilemas do primeiro amor.

Nesta página, vamos apresentar a sinopse da série de forma clara, utilizando uma estrutura simples de código HTML. O objetivo principal é demonstrar como é possível criar páginas organizadas, com linguagem acessível e conteúdo relevante, usando apenas os elementos fundamentais da programação web.

Esperamos que você aproveite a leitura e conheça um pouco mais sobre essa história que marcou — e ainda marca — a vida de tantos jovens.
